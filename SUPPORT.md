# Support

## Scope

Support covers defects in Turnstile Pass, including:

- Turnstile script and widget rendering
- Server-side verification behavior
- Automatic P&T Contact Form integration
- Configuration and documentation errors
- Compatibility issues with supported Craft CMS and PHP versions

Support does not include Cloudflare account administration, Turnstile dashboard configuration, custom form implementation, or plugin-specific integration work for other third-party form plugins.

## GitHub Issues

Open an issue at https://github.com/cdgrph/craft-turnstile-pass/issues for reproducible bugs and documentation problems.

Include the Turnstile Pass, Craft CMS, and PHP versions; concise reproduction steps; expected and observed behavior; and relevant logs with all credentials and personal data removed.

Do not submit vulnerability details through GitHub Issues. Follow the private reporting process in SECURITY.md instead.

## Email support

For questions that are not suitable for a public issue, email **hello@cdgrph.com**. We aim to send an initial response within two business days (JST).
